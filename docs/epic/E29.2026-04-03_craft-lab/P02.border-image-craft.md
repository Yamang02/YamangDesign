# E29 P02: border-image-slice Craft 구현

## 목표

- CraftLab에 `border-image-slice` 기법 섹션이 존재한다
- slice 모드 3종(stretch / repeat / round)과 fill 옵션을 라이브 예시로 비교할 수 있다 (`space` 모드는 본 Phase 범위에서 제외)
- 각 예시에 대응하는 CSS 스니펫이 표시된다

## 구현 상세

### 접근 방법

`border-image-slice`의 핵심 변수는 slice 숫자값(모서리/가장자리 분할), `border-image-repeat`(stretch·repeat·round·space), `fill` 키워드 세 가지다. 인터랙티브 슬라이더로 slice 값을 바꾸면 예시 박스와 코드 스니펫이 동기화된다.

### 변경 범위

**생성:**
- `src/app/pages/labs/CraftLab/crafts/BorderImage/BorderImageCraft.tsx`
  - slice 값 슬라이더 (1–50 범위)
  - repeat 모드 선택 (stretch / repeat / round)
  - fill 토글
  - 예시 박스 (실제 CSS `border-image` 적용)
  - 코드 스니펫 자동 생성 표시
- `src/app/pages/labs/CraftLab/crafts/BorderImage/BorderImageCraft.module.css`
- `src/app/pages/labs/CraftLab/crafts/BorderImage/index.ts`

**수정:**
- `src/app/pages/labs/CraftLab/CraftLab.tsx` — `BorderImageCraft`를 `CraftSection`에 마운트

### 사용할 border-image 소스

CSS gradient로 소스 이미지를 생성 — 외부 이미지 의존 없이 순수 CSS로 구현한다.
예: `border-image-source: repeating-linear-gradient(45deg, currentColor 0, currentColor 2px, transparent 0, transparent 50%)`
