# P01: 내비게이션 구조 재편

## 목표

`nav-categories.ts`와 라우팅을 새 3축 구조(Labs / Build / Context / Playground)로 재편한다.
새 페이지들의 빈 스텁(stub)을 만들어 후속 Phase에서 채워 넣을 수 있게 한다.

## 구현 상세

### 신규 스텁 페이지

Molecules / Organisms / Context/Shell / Context/Service — 각각 빈 `LabLayout` 래퍼에
"Coming in P0N" 안내 문구만 넣은 스텁 컴포넌트로 생성.

```tsx
// 예: src/pages/build/Molecules/Molecules.tsx
export function Molecules() {
  return (
    <LabLayout title="Molecules">
      <p style={{ color: 'var(--shell-text-secondary)' }}>
        P05에서 구현 예정
      </p>
    </LabLayout>
  );
}
```
