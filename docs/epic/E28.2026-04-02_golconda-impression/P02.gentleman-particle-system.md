# E28 / P02: 신사 파티클 시스템

## 목표

- ASCII 신사 4단계(XS/S/M/L)가 `Golconda_bg.png` 위 Canvas에서 비처럼 낙하한다.
- 각 tier별 fontSize·speed·opacity 차이로 원근감이 표현된다.
- XS·S 신사가 지붕선 아래로 내려갈 때 Pretext `layoutWithLines()`로 줄 단위 skip하여 발부터 깔끔하게 사라진다.
- 우산 인터랙션 없이 자유 낙하만 동작하는 상태가 이 Phase의 완료 기준이다.

## 구현 상세

### 접근

`@chenglou/pretext`의 **`prepareWithSegments()`** 를 각 tier의 ASCII 블록에 적용한 뒤 `layoutWithLines()`로 줄 목록을 얻는다 (브라우저 Canvas 측정 필요). `prepare()` 단독은 `layoutWithLines` 타입과 맞지 않을 수 있다.
`requestAnimationFrame` 루프에서 매 프레임 `ctx.clearRect` 후 전체 파티클을 재드로우한다.
지붕선 clipping은 Canvas clip 없이, 각 줄의 `y + lineOffset > getRooflineY(x)` 조건으로 렌더 중단한다.
