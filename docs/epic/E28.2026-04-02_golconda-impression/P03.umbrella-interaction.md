# E28 / P03: 우산 인터랙션

## 목표

- 마우스 포인터가 우산 모양 커서로 대체되어 Canvas에 렌더된다.
- 신사가 우산 반경(~80px) 내 진입 시 가장자리 방향으로 자연스럽게 밀려 흘러내린다.
- 우산 바깥으로 벗어난 신사는 다시 정상 낙하 속도로 복귀한다.
- glass 인용 카드가 Ch.2 오버레이에 표시된다.

## 구현 상세

### 접근

`mousemove` 이벤트로 커서 위치를 `useRef`에 저장하고, RAF 루프에서 Canvas에 우산을 드로우한다.
우산은 반원 호(`arc`) + 수직선(손잡이)으로 구성한다.
매 프레임 각 파티클에 대해 우산 중심과의 거리를 계산 → 반경 내이면 중심에서 멀어지는 방향으로 `dx` 보정.
`cursor: none`은 Canvas 컨테이너에 CSS로 적용한다.
