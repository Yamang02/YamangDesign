# P07: 디자인 설정 페이지 전면 리뉴얼

## 목표

`DesignSettingsLab`의 레이아웃·기능·CSS를 전면 재작성한다. 모달 잔재를 제거하고 넓어진 페이지 공간을 제대로 활용하는 구조로 전환하며, 불필요한 기능을 제거하고 누락된 기능을 보완한다.

## 제거

| 대상 | 이유 |
|---|---|
| `customSemanticPresets` (구 PaletteLab 저장소) | v2 `StoredPreset.semanticMapping`으로 대체됨. Custom 탭 이중 저장소 해소 |
| `PresetTab` 내부 footer (export/import/reset 버튼 행) | page footer로 통합 |
| `SemanticTab` 내부 footer | page footer + 탭 상단 action row로 통합 |
| `SemanticTab` h3 팔레트 제목 | 모달 헤더 잔재. 상단 컨텍스트 배지로 대체 |
| `PresetTab`의 `GlobalSettingsModal.module.css` import | 모달 CSS 의존 제거 |
| `SemanticTab`의 `SemanticMappingModal.module.css` import | 모달 CSS 의존 제거 |

## 추가

| 대상 | 설명 |
|---|---|
| 프리셋 카드 스타일 배지 | 카드에 style preset 이름 표시 (Minimal / Glass 등) |
| 프리셋 카드 시맨틱 오버라이드 인디케이터 | `semanticMapping`이 있는 프리셋에 점 표시 |
| "현재 적용됨" 배지 | 실제 적용된 프리셋과 draft를 구분 |
| 시맨틱 탭 export 버튼 | import만 있던 비대칭 구조 해소 |
| 컴포넌트 매핑: 행별 ✕ 초기화 | 토큰 단위 오버라이드 개별 제거 |
| 컴포넌트 매핑: 컴포넌트 단위 "전체 초기화" | 선택된 컴포넌트 오버라이드 일괄 제거 |
| 컴포넌트 매핑: 컬러 토큰 스와치 | "현재 값" 컬럼에 컬러값일 때 색상 ● 표시 |
| 컴포넌트 매핑: 데이터 주도 Preview | JSON `previewVariants` 필드 기반 렌더, 미지원 컴포넌트 fallback |
| CSS 변수 타입 교정 | `React.CSSProperties & Record<\`--${string}\`, string>` |

## 레이아웃

### 페이지 공통

```
[디자인 설정]
[프리셋] [시맨틱 매핑] [컴포넌트 매핑]
────────────────────────────────────────────
[탭 콘텐츠 — 탭별 전용 CSS]
────────────────────────────────────────────
[내보내기 ↓]  [가져오기 ↑]    [초기화 ↺]  [적용 ✓]
```

- `index.tsx`의 단일 page footer가 Export / Import / Reset / Apply 모두 담당
- Reset: 프리셋·시맨틱 draft 초기화 + 컴포넌트 매핑 오버라이드 초기화 포함

### 탭 1 — 프리셋 (`PresetTab.module.css` 신규)

2-column 레이아웃. 좌 = 현재 설정 편집, 우 = 프리셋 브라우저.

```
┌── 현재 설정 (300px) ──────┬── 프리셋 브라우저 (flex) ──────────────────┐
│ 컬러 팔레트               │ [Custom] [Default] [Natural] [Pop]  [검색] │
│ Primary   ● [#6366F1]    │                                            │
│ Secondary ● [#8B5CF6]    │ ┌──────────────┐ ┌──────────────┐          │
│ Accent    ● [#F59E0B]    │ │ ●●●●●         │ │ ●●●●●         │  그리드  │
│ Sub       ● [Auto   ]    │ │ Yamang        │ │ Ocean         │         │
│ Neutral   ● [#E5E7EB]    │ │ [Minimal] [✓] │ │ [Glass]  [·] │         │
│                           │ └──────────────┘ └──────────────┘          │
│ 스타일  [Minimal   ▼]    │                                            │
│ 시스템  [Default   ▼]    │ [이름________________] [프리셋으로 저장]      │
└───────────────────────────┴────────────────────────────────────────────┘
```

- 프리셋 카드: 컬러 스와치 5개 + 이름 + 스타일 배지 + 시맨틱 오버라이드 점(·)
- "현재 적용됨" 카드에 체크(✓) 배지
- Custom 탭: `userPresets`(v2)만 표시. `customSemanticPresets` 제거
- 저장 입력창: 프리셋 브라우저 하단에 상시 노출 (showSaveInput 토글 제거)

### 탭 2 — 시맨틱 매핑 (`SemanticTab.module.css` 신규)

```
팔레트: [Yamang Default]    [오버라이드 초기화 ↺]  [가져오기 ↑]  [내보내기 ↓]
┌── ScaleStepGrid (280px) ──┬── ColorUsageDiagram (flex) ─────────────────┐
│  팔레트 스케일 스텝 선택    │  시맨틱 역할 매핑 (interactive)               │
│  클릭 → 선택 토큰에 매핑    │                                              │
└───────────────────────────┴──────────────────────────────────────────────┘
```

- 상단 action row: 팔레트 컨텍스트 배지 + Reset(오버라이드만) + Import + Export
- 2패널: 탭 가용 높이를 채우도록 고정
- h3 팔레트 제목 제거

### 탭 3 — 컴포넌트 매핑 (기존 CSS 유지, 세부 보완)

```
┌── 목록 (200px) ──┬── 토큰 매핑 (1fr) ──────────────────────────┬── Preview (360px) ──┐
│ [액션]           │ [속성] [기본 토큰]     [현재 값]  [오버] [✕]│                     │
│   Button  ←선택  │ bg    --ds-action-bg  ●#6366F1  [    ] [✕]│  default variant    │
│   Input          │ text  --ds-action-text ●#fff    [    ] [✕]│  outline variant    │
│ [폼]             │  ...                                        │  ghost variant      │
│   Select         │ ─────────────── [이 컴포넌트 초기화] ───────│                     │
│ [레이아웃]        │                                             │                     │
│   Card / Badge   │                                             │                     │
└──────────────────┴─────────────────────────────────────────────┴─────────────────────┘
```

- 컬러 토큰 "현재 값": hex 앞에 ● 스와치
- 행별 ✕: 해당 토큰 오버라이드만 삭제
- "이 컴포넌트 초기화": 선택 컴포넌트 오버라이드 전체 삭제
- Preview: `component-mapping.json`에 `previewVariants: string[]` 추가, 미지원 id는 토큰 목록 텍스트 fallback
- CSS 변수 타입: `style` prop에 `React.CSSProperties & Record<\`--${string}\`, string>` 적용

## component-mapping.json 스키마 변경

```json
{
  "components": [
    {
      "id": "button",
      "label": "Button",
      "category": "action",
      "previewVariants": ["primary", "outline", "ghost"],
      "tokens": [
        { "prop": "background", "default": "--ds-action-bg", "description": "기본 배경" }
      ]
    }
  ]
}
```
