# P02: Grid Lab

## 목표

현재 정의되지 않은 그리드 토큰을 도메인 레이어에 추가하고, 컬럼 그리드 / 거터 / 마진 / 브레이크포인트를 시각적으로 탐색하는 랩 페이지를 만든다.
레이아웃 설계 시 실제 그리드가 어떻게 적용되는지 오버레이로 확인할 수 있어야 한다.

## 구현 상세

**접근 방법:**
- 도메인 토큰 먼저 정의 (컬럼 수, 거터, 마진, 브레이크포인트)
- 랩 페이지에서 그리드 오버레이를 캔버스 위에 렌더링
- 브레이크포인트별 그리드 프리뷰 (mobile / tablet / desktop 탭)
- 컬럼 수 / 거터 / 마진 수치를 슬라이더 또는 토큰 선택으로 조정

**생성 파일:**
- `src/domain/tokens/global/grid.ts` — 그리드 토큰 정의 (columns, gutter, margin, breakpoints)
- `src/app/pages/labs/GridLab/GridLab.tsx` — 메인 랩 컴포넌트
- `src/app/pages/labs/GridLab/GridLab.module.css` — 스타일
- `src/app/pages/labs/GridLab/index.ts` — re-export

**수정 파일:**
- `src/domain/tokens/global/index.ts` — grid 토큰 export 추가
- `src/app/pages/labs/index.ts` — GridLab export 추가
- `src/App.tsx` — 라우트 및 네비게이션 등록
