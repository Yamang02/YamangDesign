# P01: Spacing Lab

## 목표

`src/domain/tokens/global/spacing.ts`에 정의된 4px 기반 스케일 토큰을 시각적 블록 다이어그램으로 탐색할 수 있는 랩 페이지를 만든다.
각 스텝의 실제 크기를 눈으로 확인하고, 두 토큰 간 차이(gap diff)를 비교할 수 있어야 한다.

## 구현 상세

**접근 방법:**
- 스케일 전체를 수평 바 형태로 나열 (토큰 키 / px 값 / 시각적 블록 동시 표시)
- 두 스텝 선택 → diff 시각화 (얼마나 차이나는지 강조)
- 실사용 맥락 미리보기: padding, gap, margin 모드 전환

**생성 파일:**
