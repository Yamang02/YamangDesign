# P03: FontLab Detail 구조화

## 목표

FontLab의 Text Styles 섹션에서 카드 클릭 시 열리는 DetailPanel 내용을
`JSON.stringify` 원시 출력에서 구조화된 토큰 테이블로 교체한다.

## 구현 상세

### 현재 문제

`FontLab.tsx` DetailPanel 내부:

```tsx
// 현재: 날것의 JSON 출력
<p style={{ fontSize: 'var(--ds-text-sm)', color: 'var(--shell-text-secondary)', ... }}>
  {JSON.stringify(textStyles[selectedStyle], null, 2)}
</p>
<p style={{ fontSize: 'var(--ds-text-body-md-size)', ... }}>
  {sampleText.pangram.en}. {sampleText.numbers}
</p>
```

`JSON.stringify`는 개발 디버그 용도로 보이지만, 현재 완성된 UI에서 그대로 노출되어
사용자가 "이게 의도된 표시인가?"를 혼동하게 한다.

### 변경 방향

DetailPanel 내부를 두 영역으로 구성:

1. **토큰 속성 테이블** — 해당 TextStyle의 속성(size, weight, lineHeight, letterSpacing,
   fontFamily)을 레이블·값 쌍으로 표시. 이미 `PaletteLab`의 `PaletteDetail`과
   `Components`의 `ComponentDetail`에서 유사한 row 패턴을 사용 중이므로 동일 스타일 적용.

2. **라이브 프리뷰** — 현재 두 번째 `<p>` 유지하되, `body-md` 고정 참조를
   실제 선택된 스타일의 토큰 변수(`var(--ds-text-${selectedStyle}-size)` 등)로 교체.

```tsx
// 변경 후 구조 (FontLab.tsx 내 인라인 또는 별도 컴포넌트)
function TextStyleDetail({ name }: { name: TextStyleName }) {
  const style = textStyles[name];
  const fields = [
    { label: 'Font size', value: style.fontSize, token: `--ds-text-${name}-size` },
    { label: 'Font weight', value: String(style.fontWeight), token: `--ds-text-${name}-weight` },
    { label: 'Line height', value: String(style.lineHeight), token: `--ds-text-${name}-leading` },
    ...(style.letterSpacing ? [{ label: 'Letter spacing', value: style.letterSpacing, token: `--ds-text-${name}-tracking` }] : []),
    ...(style.fontFamily ? [{ label: 'Font family', value: style.fontFamily }] : []),
  ];
  return (
    <div>
      {/* 속성 테이블 */}
      {fields.map(({ label, value, token }) => (
        <div key={label} className={styles.detailRow}>
          <span className={styles.detailLabel}>{label}</span>
          <div className={styles.detailValue}>
            {token && <code className={styles.detailToken}>{token}</code>}
            <span>{value}</span>
          </div>
        </div>
      ))}
      {/* 라이브 프리뷰 */}
      <p style={{
        fontSize: `var(--ds-text-${name}-size)`,
        fontWeight: `var(--ds-text-${name}-weight)`,
        lineHeight: `var(--ds-text-${name}-leading)`,
        color: 'var(--shell-text-primary)',
        marginTop: 'var(--ds-spacing-4)',
      }}>
        {sampleText.pangram.en}
      </p>
    </div>
  );
}
```

영향 범위: `FontLab.tsx`, `FontLab.module.css` (detailRow/detailLabel/detailValue/detailToken 스타일 추가)
