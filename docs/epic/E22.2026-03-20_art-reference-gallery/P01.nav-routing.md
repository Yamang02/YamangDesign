# P01: Nav 통합 및 라우팅

## 목표

"Art" 카테고리를 헤더 네비게이션에 추가하고, MonetWaterLilies 페이지로의 라우팅을 연결한다.
이 Phase 완료 후 Nav에서 Art > Water Lilies를 클릭하면 빈 페이지(placeholder)라도 정상 렌더링된다.

## 구현 상세

**접근 방법:**
기존 Labs/Build/Context와 동일한 방식으로 `navCategories` 배열에 `art` 카테고리를 추가하고,
`App.tsx`의 `PageName` 타입 및 `renderPage()` switch에 케이스를 연결한다.

**변경 파일:**
- `src/app/config/nav-categories.ts` — `art` 카테고리 + `monet-water-lilies` 항목 추가
- `src/App.tsx` — `PageName`에 `'monet-water-lilies'` 추가, `renderPage()` switch 케이스 추가, `MonetWaterLilies` import
- `src/app/pages/art/MonetWaterLilies/index.ts` — 신규 생성 (export)
- `src/app/pages/art/MonetWaterLilies/MonetWaterLilies.tsx` — 신규 생성 (placeholder)
- `src/app/pages/index.ts` — `MonetWaterLilies` export 추가

**주요 결정:**
- 아이콘: `'frame'` (material) 또는 적절한 대체 아이콘 사용
- Art 카테고리는 `items` 배열을 가진 드롭다운 형태
