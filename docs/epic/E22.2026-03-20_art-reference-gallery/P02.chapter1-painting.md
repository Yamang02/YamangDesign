# P02: Chapter 1 — The Painting

## 목표

모네 원화 이미지를 직접 사용하는 히어로 구간을 구현한다.
이미지가 뷰포트 배경에 sticky로 고정되고, 그 위에 글라스모피즘 패널들이 레이어링된다.
스크롤 진행에 따라 이미지에 blur가 적용되며 Chapter 2로의 시각적 전환이 발생한다.

## 구현 상세

**접근 방법:**
`MonetWaterLilies.tsx` 안에 Chapter 1 전용 서브 컴포넌트들을 `components/` 폴더로 분리한다.
모든 스타일은 `MonetWaterLilies.module.css`와 각 컴포넌트의 CSS Module에서 완결한다.
JS scroll listener로 scroll progress를 계산하여 이미지 blur 값을 동적으로 조정한다.

**변경 파일 (신규 생성):**
- `src/app/pages/art/MonetWaterLilies/MonetWaterLilies.tsx` — 메인 페이지, Chapter 1/2 조합
- `src/app/pages/art/MonetWaterLilies/MonetWaterLilies.module.css` — 페이지 레이아웃, 챕터 전환
- `src/app/pages/art/MonetWaterLilies/components/HeroStage.tsx` — sticky 이미지 컨테이너
- `src/app/pages/art/MonetWaterLilies/components/HeroStage.module.css`
- `src/app/pages/art/MonetWaterLilies/components/ArtworkInfoCard.tsx` — 작품 정보 glass 패널
- `src/app/pages/art/MonetWaterLilies/components/ArtworkInfoCard.module.css`
- `src/app/pages/art/MonetWaterLilies/components/PaletteSwatchBar.tsx` — 색상 스와치 + 복사
- `src/app/pages/art/MonetWaterLilies/components/PaletteSwatchBar.module.css`

**글라스모피즘 CSS 기준:**
- `background: rgba(255, 255, 255, 0.08)`
- `backdrop-filter: blur(20px)`
- `border: 1px solid rgba(255, 255, 255, 0.2)`
- `border-radius` → `--ds-border-radius-lg` 참조
- 간격/타이포 → `--ds-spacing-*`, `--ds-text-*` 참조

**HeroStage 구조:**
```
.heroStage (min-height: 200vh, position: relative)
  .stickyBg (position: sticky, top: 0, height: 100vh)
    img (object-fit: cover, filter: blur(Xpx) — JS 제어)
  .floatingPanels (position: absolute, inset)
    ArtworkInfoCard (우측 하단)
    PaletteSwatchBar (좌측 하단)
```
