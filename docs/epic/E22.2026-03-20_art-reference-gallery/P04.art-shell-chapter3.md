# P04: ArtShell + Chapter 3 — The Application

## 목표

모든 아트 페이지에 공통 적용되는 shell 레이아웃을 신설하고,
모네 팔레트 × DS 토큰을 실제 컴포넌트에 적용하는 Chapter 3를 구현한다.

## 구현 상세

**ArtShell (`_shared/ArtShell.tsx`):**
- 좌측 sticky 사이드바 — 챕터 네비게이션 전용 (IntersectionObserver 기반 active 추적)
- 우측 main 영역 — 각 작품 페이지의 챕터 콘텐츠 children으로 수신
- CSS Grid `140px 1fr` 레이아웃
- `ArtChapter[]` props로 챕터 목록 주입 → 재사용 가능

**Chapter 1 정보 표시 변경:**
- `ArtworkInfoCard` (glass panel) → `MuseumLabel` 로 교체
- 위치: 이미지 우측 상단, 미술관 플라카드 스타일 (텍스트만, 배경 없음)
- 내용: 제목 / 작가 / 기법·연도

**Chapter 3 (`ComponentApplication.tsx`):**
모네 팔레트(MONET_PALETTE)에서 추출한 색상을 CSS custom property 인라인 오버라이드로
DS 컴포넌트에 주입하는 방식으로 구현. ThemeProvider는 건드리지 않음.

구현 예정 컴포넌트 블록:
- 타이포그래피 — 모네 팔레트 색상 적용한 heading/body 텍스트 조합
- 버튼 세트 — primary(Deep Green), secondary(Teal), accent(Mustard) 변형
- 컬러 카드 — 각 팔레트 색상을 배경으로 한 정보 카드
- 배지/태그 — Lavender/Water Blue 계열
