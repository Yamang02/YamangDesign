# P03: token-section-refactor

## 목표
3벌 중복된 TokenDetail 컴포넌트를 공유 `TokenSection`으로 통합한다.

## 구현 상세

### 공유 컴포넌트 생성
**파일**: `src/components/TokenSection/TokenSection.tsx` (신규)
**파일**: `src/components/TokenSection/index.ts` (신규)

```ts
export interface TokenSectionProps {
  tokens: Array<{ token: string; label?: string }>;
}
```

내부 구현:
- `getTokenCategory(token)` — 6개 슈퍼셋 분류:
  `color` / `spacing` (spacing, radius, z-) / `typography` (font, text-, weight, leading) / `size` (size, border) / `shadow` (shadow, transition) / `other`
- prefix strip: `--ds-`, `--sys-`, `--shell-` 모두 제거
- color swatch: `token.includes('color')` && 값이 `#` / `rgb` / `hsl`로 시작
- `useEffect`로 `getComputedStyle(document.documentElement)` 읽어 실제 값 표시
- 빈 카테고리 자동 스킵

스타일은 P02에서 수정한 `Components.module.css`의 토큰 클래스를 그대로 import해서 사용하거나, 컴포넌트 전용 CSS module로 분리.
