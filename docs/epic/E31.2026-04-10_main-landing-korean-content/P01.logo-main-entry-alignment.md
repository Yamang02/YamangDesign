# E31 P01: Logo Main Entry Alignment

## 목표
- Header 로고의 클릭 동작이 `main` 진입점으로 고정되어, 어느 페이지에서든 동일한 복귀 경험을 제공한다.

## 구현 상세
- 검증 대상:
  - `src/App.tsx`의 초기 페이지 상태(`useState<PageName>('main')`)
  - `Header`의 `onLogoClick` 연결(`setPage('main')`)
- 필요 시 수정 범위:
  - `src/App.tsx`
  - (추가 변경 필요 시) `src/app/components/Header/*`

## 체크리스트
- [x] 초기 렌더 시 `Main` 페이지가 표시된다.
- [x] 로고 클릭 시 현재 페이지와 무관하게 `Main`으로 이동한다.
- [x] 기존 콘텐츠 랜딩(`landing`) 접근 경로는 유지된다.
