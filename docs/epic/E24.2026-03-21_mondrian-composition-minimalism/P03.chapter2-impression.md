# P03: Ch.2 — The Impression

## 목표

마우스가 지나간 격자 셀에 muted 색이 채워지는 인터랙션 + 인용구 카드. 모네(Canvas ripple), 마티스(bold typography)와 완전히 다른 방식.

- `src/app/pages/art/MondriComposition/components/GridBackground.module.css`
- `src/app/pages/art/MondriComposition/components/MondriGrid.tsx` — N×M div 셀 인터랙션
- `src/app/pages/art/MondriComposition/components/MondriGrid.module.css`

- `src/app/pages/art/MondriComposition/components/MondrianQuote.module.css`

**수정 파일:**
- `src/app/pages/art/MondriComposition/MondriComposition.tsx` — Ch.2 섹션 추가

**GridBackground:**
- `background-image: linear-gradient(...)` 또는 CSS `border` 기반 격자선
- 색상: `rgba(28, 28, 28, 0.08)` (Grid Black 8% 투명도)
- 배경: `#F5F2EC` (Canvas White)

**MondriGrid:**
- 셀 수: 가로 16 × 세로 10 (반응형 고려, 모바일 8×6)
- 마우스 `onMouseEnter` per cell → 해당 셀에 muted 색 적용
- 색상 풀: `['#7A9BBF', '#C47A6A', '#D4C06A', '#B0ADA8']` 중 랜덤 or 순환
- `transition: background-color 0.3s ease`
- 마우스가 떠나도 색 유지 (캔버스처럼 그려진 상태 유지)
- 리셋 버튼 (선택): 더블클릭 또는 버튼으로 초기화

**MondrianQuote:**
- 배경: `rgba(245, 242, 236, 0.85)`, border: `1px solid #1C1C1C`
- 인용구: *"The position of the artist is humble. He is essentially a channel."*
- 출처: — Piet Mondrian
- 폰트: serif, 절제된 크기
