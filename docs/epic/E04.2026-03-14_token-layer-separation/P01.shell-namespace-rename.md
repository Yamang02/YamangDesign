# P01: Shell 네임스페이스 Rename

## 목표

`--ui-*` → `--shell-*`, `data-ui` → `data-shell`, `ui-variables.css` → `shell-variables.css`로 일괄 rename하여 앱 셸 레이어 네이밍을 확정한다.

## 구현 상세

순수 기계적 rename. 동작 변경 없음.

**변경 대상:**
1. `src/styles/ui-variables.css` → `src/styles/shell-variables.css` (파일명 변경)
2. 파일 내부 `:root`, `[data-ui]` 셀렉터 → `[data-shell]` 으로 변경
3. `--ui-` 접두사 → `--shell-` 로 전체 치환
4. `src/styles/index.ts` 등 import 경로 업데이트
5. TSX 파일의 `data-ui` 속성 → `data-shell`
6. CSS 모듈의 `[data-ui]` 셀렉터 → `[data-shell]`
7. `ThemeProvider.tsx`의 `[data-ui]` 참조 업데이트
8. TypeScript 타입 정의(`src/tokens/ui/`) 파일명/내용 업데이트

**영향 파일 (예상):**
- `src/styles/ui-variables.css` (rename + 내부 치환)
- `src/index.css` (ui-variables.css → shell-variables.css import 경로)
- `src/themes/ThemeProvider.tsx` (현재 코드에 `[data-ui]` 문자열 없음 — 주석/문서만 있을 경우 정리)
- `src/components/Header/Header.tsx`, `HeaderNav.tsx`
- `src/components/Navigation/Navigation.tsx`
- `src/components/DetailPanel/DetailPanel.tsx`
- `src/components/GlobalSettings/GlobalSettingsModal.tsx`
- `src/components/Footer/Footer.tsx`

- `src/pages/layouts/PaletteLab/ScaleSelectionModal.tsx`, `SemanticMappingModal.tsx`
- `src/components/ComponentDetailModal/ComponentDetailModal.tsx`
- **CSS 모듈 (`--ui-*` 사용):** `Header.module.css`, `HeaderNav.module.css`, `Navigation.module.css`, `NavDropdown.module.css`, `DetailPanel.module.css`, `GlobalSettings/GlobalSettingsModal.module.css`, `Tooltip.module.css`, `Footer.module.css`, `LabLayout.module.css`, `LabOverview.module.css`, `ThemeTabNavigation.module.css`, `ThemeSearchBar.module.css`, `ScaleStepGrid.module.css`, `ScaleSelectorPanel.module.css`, `ScaleGuide.module.css`, `EmptyCategory.module.css`, `ColorUsageDiagram.module.css`, `ScaleSelectionModal.module.css`, `SemanticMappingModal.module.css`, `FontOverviewDiagram.module.css`, `StyleOverviewDiagram.module.css`, `ComponentDetailModal.module.css`
