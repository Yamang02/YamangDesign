# P08: Token Lab Overview 신설 + Detail Panel + CSS 정리

## 목표

Token Lab의 복잡함을 줄이기 위해:
1. 기존 Architecture 섹션을 `TokenOverviewDiagram`으로 대체 (카테고리 설명 인라인 포함)
2. 토큰 행 클릭 시 Detail Panel에서 해석 체인 표시
3. 미사용 CSS 제거로 코드 정리

## 구현 상세

### 1. TokenOverviewDiagram 컴포넌트

신규 파일: `src/pages/labs/TokenLab/TokenOverviewDiagram.tsx`

StyleOverviewDiagram 패턴을 따르되, 3-layer 구조 + 카테고리별 설명 포함.

```
[Shell Layer]              [DS Layer]                  [Sys Layer]
고정 라이트, 테마 무관      테마 반응형 (ThemeProvider)  상태 색상, 테마 무관
─────────────────────      ──────────────────────────  ──────────────────
bg    배경 계층             color   팔레트 × 스타일       error   에러 상태
text  텍스트 계층           spacing 간격 스케일           warning 경고 상태
border 경계선              typography 서체 정의           success 성공 상태
action 인터랙션 색상        motion  전환 속도·이징         info    안내 상태
shadow 깊이 표현
overlay 오버레이·툴팁
badge 배지 색상
```

- `LabOverview` 래퍼 사용 (StyleLab 동일 패턴)
- 기존 `LayerDiagram` / Architecture 섹션 제거

### 2. Token Lab Detail Panel

`TokenLab.tsx`에 `useState<string | null>(selectedToken)` 추가.

행 클릭 가능하도록 `TokenTableRow`에 `onClick` prop 추가:

```tsx
<tr onClick={() => onSelect?.(token)} style={{ cursor: 'pointer' }}>
```

Detail Panel 컨텐츠:
```
제목: --shell-bg-base
카테고리: 배경 (bg)
레이어: Shell — 고정 라이트, 테마 무관

현재 값
  #ffffff

해석 체인
  --shell-bg-base
    → --ds-color-bg-base
      → #ffffff
```

- `useCssVar`로 현재 값 읽기 (기존 훅 재활용)
- 체인은 `getComputedStyle` + CSS var 참조 순서로 표시
- `<DetailPanel>` 컴포넌트 재활용 (StyleLab 동일 패턴)
