# P06: StyleLab Overview 개편

## 목표

`StyleOverviewDiagram`의 "조정 가능한 요소" 섹션을 Property Matrix의 전체 속성(5개 그룹)과 정렬한다. 현재 Shadow / Border만 기술되어 있어 Overview와 Matrix 간 정보 불일치가 있다.

## 구현 상세

### 추가할 속성 그룹

| 그룹 | 기존 | 변경 |
|------|------|------|
| Elevation | Shadow (부분) | shadow scale 전체 (none/sm/md/lg/xl/inset) |
| Stroke | Border (부분) | width + colorStrategy |
| Material | ❌ | backdropFilter, backgroundAlpha (glass/blur) |
| Filter | ❌ | element (CSS filter) |
| Spatial | ❌ | perspective (3D depth) |

### 변경 파일

- `src/pages/labs/StyleLab/StyleOverviewDiagram.tsx`
  - `styleProperties` 배열을 5개 그룹으로 확장
  - `ShadowScaleDiagram` → Elevation 섹션으로 흡수 또는 리네임
  - `StyleComparisonDiagram` 유지 (프리뷰 역할)
- `src/pages/labs/StyleLab/StyleOverviewDiagram.module.css`
  - 새 속성 카드에 필요한 스타일 추가 (기존 패턴 재사용)

### 설계 원칙

- 각 속성 카드: 이름 + 한 줄 설명 + 대표 값 태그들 + CSS 변수명
- Material / Filter / Spatial은 일부 스타일에서만 활성화됨을 시각적으로 표기 (예: 값 태그에 `—` 표시)
