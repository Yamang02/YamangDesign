# P07: StyleLab Property Matrix 셀 값 축약 + 섹션 너비 통일

## 목표

Property Matrix 셀의 긴 값(shadow CSS string 등)을 축약하여 한 줄 표시를 보장하고,
StyleLab 섹션 내 컨텐츠 컨테이너 너비를 통일한다.

## 구현 상세

### 1. Property Matrix 셀 값 축약

**문제**: 셀에 `box-shadow: 0px 4px 8px rgba(0,0,0,0.1)` 같은 전체 CSS 문자열이 들어가
`word-break: break-all`로 줄바꿈 발생.

**해결**: `getValue` 함수를 수정해 의미 있는 키 이름만 반환.

| 행 | 기존 반환값 | 변경 후 |
|----|------------|---------|
| elevation (sm) | `0px 4px 8px…` (full CSS) | `shadow-sm` |
| elevation (inset) | full CSS | `inset` 또는 `shadow-inset` |
| material blur | `blur(8px) saturate(…)` | `8px` (blur 수치만) |
| material alpha | `0.8` | 유지 (이미 짧음) |
| stroke width | `1px` | 유지 |
| stroke strategy | `shift` 등 | 유지 |
| filter | `none` 또는 CSS | `none` / `saturate(1.2)` 등 짧게 |
| spatial | perspective 값 | 유지 또는 `none` |

### 2. Property Matrix 폰트 크기

셀(`propertyMatrixCell`)에 container query 기반 자동 축소:

```css
.propertyMatrixCellWrap {
  container-type: inline-size;
}
.propertyMatrixCell {
  font-size: clamp(9px, 1.4cqi, 11px);
  white-space: nowrap;
}
```

값 축약 후에는 대부분 한 줄에 들어오므로 폰트 크기 조정은 최소화.

### 3. 섹션 컨텐츠 너비 통일

**문제**: `LabSection`의 `sectionContent`가 `display: flex; flex-wrap: wrap`이라
단일 자식도 content 크기로 수축할 수 있음. 각 섹션 컨텐츠가 너비가 다르게 보임.

**해결**: StyleLab의 컨텐츠 래퍼에 `width: 100%` 추가.

변경 파일:
- `StyleLab.module.css`
  - `.comparisonWrapper` → `width: 100%` 추가
  - `.propertyMatrixWrap` → 음수 마진(`margin: 0 calc(-1 * var(--ds-spacing-4))`) 제거,
    `width: 100%`로 교체

## 체크리스트

- [ ] `PropertyMatrix`의 `getValue` 함수들 값 축약 (elevation → shadow key명)
- [ ] `propertyMatrixCell`에 `white-space: nowrap` 또는 `clamp` font-size 적용
- [ ] `.comparisonWrapper`에 `width: 100%` 추가
- [ ] `.propertyMatrixWrap` 음수 마진 제거 및 너비 통일
