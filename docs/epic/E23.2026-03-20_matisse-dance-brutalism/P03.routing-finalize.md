# P03: 라우팅 통합 + 마무리

## 목표

MatisseDanceII 페이지를 앱 라우팅에 연결하고, TypeScript 에러 없음 + 전체 동작을 최종 검증한다.

## 구현 상세

### 수정 파일

**`src/app/config/nav-categories.ts`**
```ts
{ id: 'matisse-dance-ii', label: 'La Danse II', icon: 'image' }
// art 카테고리 items 배열에 추가
```

**`src/App.tsx`**
```ts
// PageName 타입에 추가
| 'matisse-dance-ii'

// renderPage() switch에 추가
case 'matisse-dance-ii':
  return <MatisseDanceII />;
```

**`src/app/pages/index.ts`**
```ts
export { MatisseDanceII } from './art/MatisseDanceII';
```
