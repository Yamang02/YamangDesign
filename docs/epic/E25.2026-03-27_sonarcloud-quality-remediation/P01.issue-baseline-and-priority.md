# P01: 이슈 베이스라인 확정 + 우선순위 수립

## 목표

현재 Sonar 이슈를 규칙/심각도/파일 단위로 재현 가능한 기준선으로 고정하고,
다음 Phase에서 바로 수정 가능한 우선순위 백로그를 만든다.

## 구현 상세

- 기준 데이터: `sonar-issues.json` (385건)
- 분석 포인트:
  - Severity 분포 (`CRITICAL`, `MAJOR`, `MINOR`, `INFO`)
  - Type 분포 (`BUG`, `VULNERABILITY`, `CODE_SMELL`)
  - 빈도 상위 규칙 (`typescript:S6759`, `typescript:S3358`, `typescript:S4325` 등)
  - 이슈 집중 파일 (`TokenLab.tsx`, `PaletteLab.tsx`, `ThemeProvider.tsx`, `Select.tsx`, 스크립트 파일)
- 변경/참조 파일:
  - `docs/epic/E25.2026-03-27_sonarcloud-quality-remediation/Readme.md`
  - `docs/epic/E25.2026-03-27_sonarcloud-quality-remediation/P01.issue-baseline-and-priority.md`
