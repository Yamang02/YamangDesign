# P02: Ch.2 StarryNightFrame

## 목표

- Ch.2 섹션에서 Van Gogh 자화상이 화면 중앙에 표시되고, 그 외곽 프레임 영역에 소용돌이 Canvas 붓터치 애니메이션이 흘러간다.
- 마우스 X로 frame inset 비율(sliceValue), Y로 frame 두께(frameSize)가 실시간 조정된다.
- HUD 레이블이 `border-image-slice: {sliceValue}` / `border-width: {frameSize}px` 수치를 표시한다.
- 마우스가 컴포넌트 밖으로 벗어나면 기본값으로 ease-out 복귀한다.

## 구현 상세

### 접근

`StarryNightFrame` 컴포넌트는 `position: relative` wrapper 위에 동일 크기의 `<canvas>`를 `position: absolute`로 올린다.

```
[ wrapper ]
  ├── <img> 자화상 (중앙 고정)
  ├── <canvas> (wrapper 전체 덮음, pointer-events: none)
  │     └── rAF 루프: 프레임 영역만 소용돌이 stroke 드로잉
  │           clearRect(imageRect) 로 자화상 영역은 투명 유지
  └── <div> HUD (우하단 고정)
        "border-image-slice: {sliceValue}"
        "border-width: {frameSize}px"
```

**붓터치 드로잉 알고리즘**
- Starry Night 팔레트 6색을 순환하며 `strokeStyle` 지정
- `bezierCurveTo`로 소용돌이 곡선 경로 생성 (frame 영역 안에서만)
- `lineWidth` 3~8px 랜덤 → 임파스토 질감
- 매 프레임 전역 `offset` 증가 → 소용돌이가 시계방향으로 흘러감
- frame 영역 = wrapper rect - imageRect (이미지 기준 inset은 frameSize)

**마우스 인터랙션**
- `mousemove` X: `sliceValue = Math.round(map(x, 0, wrapperWidth, 20, 120))`
- `mousemove` Y: `frameSize = Math.round(map(y, 0, wrapperHeight, 40, 120))`
- `mouseleave`: `sliceValue`, `frameSize` → 기본값(60, 60)으로 `requestAnimationFrame` ease-out

**sliceValue 시각 반영**
- `sliceValue`가 클수록 frame inset이 이미지 쪽으로 깊이 들어옴
  → `imageInset = frameSize * (sliceValue / 120)`으로 `clearRect` 영역 계산

## 참고 결정사항 (구현 진화)

초기 Phase 초안은 마우스 HUD·소용돌이 stroke rAF 중심이었으나, Ch.2는 **CSS border-image vs Canvas wavy ring 비교**와 **안정적인 텍스처 링**에 맞춰 다음으로 정리되었다.

- `StarryNightFrame`: `variant`로 `css` | `canvas` 분기. Canvas variant는 `StarryNightCanvasBorder` + 자화상 `object-fit: cover`(`portraitCrop`).
- `StarryNightCanvasBorder`: 외곽 물결 경로 + 내부 라운드 홀을 `clip('evenodd')`로 링만 그린 뒤 단색·텍스처 덮기; 내부 치수가 깨지면 상·하·좌·우 밴드 폴백. 물결 외곽은 얇은 stroke로 장식.
- 에픽 Readme의 **구현 참고 (Ch.2 프레임)** 와 동일한 트레이드오프·운영 주의를 따른다.
