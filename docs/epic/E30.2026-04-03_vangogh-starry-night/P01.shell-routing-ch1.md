# P01: 셸 + 라우팅 + Ch.1

## 목표

- `nav-categories.ts`, `App.tsx`, `pages/index.ts`에 VanGoghStarryNight 라우팅이 등록되어 `/art/starry-night`로 접근 가능하다.
- Ch.1 HeroStage에서 Starry Night 원화, MuseumLabel, PaletteSwatchBar가 렌더된다.
- `starry-night-palette.ts`에 8색 팔레트 상수가 정의되어 있다.

## 구현 상세

### 접근

기존 Golconda / Matisse 페이지 패턴을 그대로 따른다. `ArtHeroStage variant="minimal"` + `ArtShell theme="dark"` 사용.

### 변경 범위

**신규 생성**
- `src/app/pages/art/VanGoghStarryNight/VanGoghStarryNight.tsx`

- `src/app/pages/art/VanGoghStarryNight/starry-night-palette.ts`
- `src/app/pages/art/VanGoghStarryNight/index.ts`
- `src/app/pages/art/VanGoghStarryNight/components/` (디렉토리, 파일 없음)

**수정**
- `src/app/config/nav-categories.ts` — art 카테고리에 `{ id: 'starry-night', label: 'Starry Night', page: 'VanGoghStarryNight' }` 추가
- `src/App.tsx` — `PageName` 타입에 `'VanGoghStarryNight'` 추가, `renderPage()` switch 케이스 추가
