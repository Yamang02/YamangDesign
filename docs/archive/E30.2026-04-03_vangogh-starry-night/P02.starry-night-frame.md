# P02: Ch.2 StarryNightFrame

## 목표

- Ch.2에서 Van Gogh 자화상이 두 패널 모두에 동일한 레이아웃(`portraitCrop` 3:4)으로 표시된다.
- 좌측 패널은 CSS `border-image` 9-slice(`seamless_bg.png`), 우측은 Canvas 물결 링(`StarryNightCanvasBorder`)으로 테두리를 그린다.
- 공통 폭 슬라이더·좌측 `border-image-repeat` 셀렉트·우측 motion 셀렉트로 비교 탐구가 가능하다.
- Canvas 변형에서 별·달·나무 데칼은 링 영역 밖으로 비치지 않도록 `starryNightRingPath` 기반 `clip-path`로 제한되고, 자화상 위 레이어 순서가 맞는다.

## 구현 상세

### 접근

- `StarryNightFrame` — `variant`: `css` | `canvas`.
- `StarryNightCanvasBorder` — `readPaddingPx`, `traceOuterWavyPath`, `clip('evenodd')` 링 채우기·물결 stroke.
- `starryNightRingPath.ts` — 캔버스 경로와 동일한 SVG path d → `clip-path: path(evenodd, …)` for `canvasDecalClip`.
- `VanGoghStarryNight.tsx` — Ch.2 비교 그리드·컨트롤.

### 변경 범위 (요지)

- `components/StarryNightFrame.tsx`, `StarryNightFrame.module.css`
- `components/StarryNightCanvasBorder.tsx`, `StarryNightCanvasBorder.module.css`
- `components/starryNightRingPath.ts` (공유 기하)
- `VanGoghStarryNight.tsx`, `VanGoghStarryNight.module.css`
