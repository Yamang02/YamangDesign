# P03: Motion Lab

## 목표

`src/domain/tokens/global/motion.ts`에 정의된 duration, easing, stateLayer 토큰을 실제 CSS 애니메이션으로 체감할 수 있는 랩 페이지를 만든다.
이징 커브의 차이와 duration 스텝을 눈으로 비교할 수 있어야 한다.

## 구현 상세

**접근 방법:**
- **Duration 섹션:** 동일한 이징에서 각 duration 스텝(instant/fast/normal/slow/slower)을 동시에 실행해 속도 차이 체감
- **Easing 섹션:** 동일한 duration에서 각 easing 커브(linear/easeIn/easeOut/easeInOut/productive/expressive)를 동시에 실행
- **커브 시각화:** SVG로 cubic-bezier 커브를 그려 이징 형태 확인
- **Replay 버튼:** 애니메이션 재실행 트리거
- **stateLayer 섹션:** hover/pressed/focus/selected/disabled opacity 값을 색상 블록으로 시각화

**생성 파일:**
- `src/app/pages/labs/MotionLab/MotionLab.tsx` — 메인 랩 컴포넌트
- `src/app/pages/labs/MotionLab/MotionLab.module.css` — 스타일
- `src/app/pages/labs/MotionLab/index.ts` — re-export

**수정 파일:**
