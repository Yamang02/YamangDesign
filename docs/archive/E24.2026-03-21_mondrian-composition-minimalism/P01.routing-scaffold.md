# P01: 라우팅 + 폴더 구조

## 목표

몬드리안 페이지가 앱에 연결되어 라우팅으로 진입 가능한 상태.
컴포넌트 구현 없이 빈 페이지가 렌더되면 완료.

## 구현 상세

**생성 파일:**
- `src/app/pages/art/MondriComposition/MondriComposition.tsx` — 빈 페이지 컴포넌트
- `src/app/pages/art/MondriComposition/MondriComposition.module.css`
- `src/app/pages/art/MondriComposition/components/` — 빈 디렉토리
- `src/app/pages/art/MondriComposition/mondri-palette.ts` — 팔레트 상수
- `src/app/pages/art/MondriComposition/index.ts` — export

**수정 파일:**
- `src/app/config/nav-categories.ts` → art 카테고리에 MondriComposition 항목 추가
- `src/App.tsx` → `PageName` 타입 + `renderPage()` switch 케이스 추가
- `src/app/pages/index.ts` → export 추가

**팔레트 상수 (mondri-palette.ts):**
