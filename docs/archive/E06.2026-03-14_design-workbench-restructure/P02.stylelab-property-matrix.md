# P02: Style Lab 재설계 — Property Matrix + Glassmorphism

## 목표

Style Lab을 "preset 비교 도구"에서 "스타일이 어떤 토큰으로 정의되는지 학습하는 도구"로 재설계한다.
`StyleDefinition`의 모든 슬롯을 표로 노출하고, glassmorphism 프리셋으로 material 슬롯을 실증한다.

## 구현 상세

### 새 섹션 구조

```
StyleLab
├── Overview          현재 스타일 전체 다이어그램 (기존 StyleOverviewDiagram 유지)
├── Property Matrix   ← 신규: 어떤 슬롯이 이 ism을 만드는가
├── Shadow            기존 shadow comparison (기준 컨텍스트 래퍼 추가 — Readme §비교 원칙)
├── Components        기존 component comparison (동일)
└── DetailPanel       기존 StyleDetail (그대로 유지)
```

### Property Matrix 섹션

각 `StyleDefinition` 슬롯을 행으로, 각 프리셋을 열로 배치한 표.
현재 선택된 스타일 행이 강조(selected state).

```
속성 차원        | minimal      | neumorphism    | brutalism    | glassmorphism
─────────────────┼──────────────┼────────────────┼──────────────┼──────────────
elevation (sm)   | drop ↓       | bilateral ↕    | hard-offset → | drop ↓
elevation (inset)| inset ↓      | inset ↕        | inset →      | inset ↓
stroke width     | 1px          | 0              | 3px          | 1px
stroke strategy  | palette      | transparent    | palette      | fixed (rgba)
material blur    | —            | —              | —            | blur(12px)
material alpha   | —            | —              | —            | 0.6
filter           | —            | —              | —            | —
spatial          | —            | —              | —            | —
```

각 셀은 실제 CSS 값을 짧게 표시. `—`는 해당 슬롯이 미설정임을 표시.
셀 클릭 시 해당 슬롯의 CSS 변수명 + 현재 값을 토글 팝오버로 표시.

**컴포넌트 구조:**
```tsx
function PropertyMatrix({ presets, activeStyle, onStyleSelect }) {
  // 행: PROPERTY_DIMS (elevation.1, elevation.inset, stroke.width, ...)
  // 열: presets (minimal, neumorphism, brutalism, glassmorphism)
  // 셀: resolved 값 표시 + activeStyle 행 하이라이트
}
```

### Comparison 섹션 — 기준 컨텍스트 래퍼 적용

기존 Shadow/Component Comparison 섹션에 `fixedBaseContext` 래퍼 추가:

```tsx
// StyleLab.tsx의 Comparison 섹션
const fixedPaletteVars = getPaletteVariablesFromDefinition(defaultPaletteDefinition);

<div style={fixedPaletteVars}>  {/* 팔레트 고정 래퍼 */}
  {comparisonPresets.styles.map((styleName) => (
    <ComparisonCard styleVars={getStyleVariables(styleName, bgColor)} ...>
      {/* shadow/component demo */}
    </ComparisonCard>
  ))}
</div>
```

Shadow demo 박스는 `var(--ds-shadow-*)` 참조로 변경 (E05 P02 내용 흡수).

### Glassmorphism 프리셋 추가

```typescript
// src/styles/presets/glassmorphism.ts
export const glassmorphismStyle: StyleDefinition = {
  name: 'glassmorphism',
  elevation: {
    create: () => ({
      0: 'none',
      1: '0 2px 8px rgba(0,0,0,0.12)',
      2: '0 8px 20px rgba(0,0,0,0.15)',
      3: '0 16px 32px rgba(0,0,0,0.18)',
      4: '0 24px 48px rgba(0,0,0,0.2)',
      inset: 'inset 0 1px 0 rgba(255,255,255,0.4)',
    }),
  },
  stroke: {
    width: '1px',
    style: 'solid',
    colorStrategy: 'fixed',
    fixedColor: 'rgba(255,255,255,0.25)',
  },
  material: {
    backdropFilter: 'blur(12px) saturate(1.5)',
    backgroundAlpha: 0.55,
    backgroundBlendMode: 'normal',
  },
};
```

`StyleName` 타입에 `'glassmorphism'` 추가.
`stylePresets`에 등록. `comparisonPresets.styles`에 추가.

**Glassmorphism이 제대로 보이려면 배경이 필요하다.** Style Lab의 Overview/Comparison 섹션 배경에
그라데이션 또는 색상 이미지를 조건부로 적용 (glassmorphism 선택 시에만).
