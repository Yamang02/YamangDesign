# P03: Shell 컨텍스트 색상 토큰 교정

## 목표

`[data-shell]` 컨텍스트 내부 CSS 파일에서 색상 계열 `--ds-color-*`를 `--shell-*` 직접 참조로 교체한다. 누락된 `--shell-*` 토큰은 `shell-variables.css`에 추가한다.

## 구현 상세

**원칙:**
```
[data-shell] 컨텍스트 안
  색상 (bg, text, border, shadow) → --shell-* 직접 참조
  프리미티브 (spacing, radius, typography, motion) → --ds-* 그대로 유지
  preview 복원 영역 ([data-context="preview"]) → --ds-* 그대로 유지 (복원 로직 필수)
```

**누락 토큰 추가 (shell-variables.css에 추가 필요 예상):**
- `--shell-bg-muted` (LabLayout `.tabBar` 배경, EmptyCategory, ScaleGuide, ColorUsageDiagram, StyleOverviewDiagram, FontOverviewDiagram 등에서 `var(--ui-bg-muted)` 사용 중. 현재 `ui-variables.css`에 정의 없음 → `--shell-bg-surface`와 동일값 또는 별칭으로 추가)
- `--shell-border-focus` (TOC 활성 항목 구분선)
- 기타 grep 결과에서 발견되는 `--ds-color-*` → 대응 shell 토큰 없는 것들
