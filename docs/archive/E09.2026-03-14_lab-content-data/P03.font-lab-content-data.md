# P03: Font Lab 콘텐츠 데이터화

## 목표

Font Lab 오버뷰의 타이포그래피 속성·시맨틱 역할·타입 스케일 등을 JSON으로 분리하고, `FontOverviewDiagram` 및 관련 UI가 해당 데이터를 사용하도록 한다.

## 구현 상세

### 1. 데이터 파일

- `src/content/labs/font-lab/overview.json`:
  - **typographyProperties**: `{ key, label, description, scale[], cssVar }[]`
  - **semanticRoles**: `{ role, styleName, description, example }[]`
  - **sizeScale** (또는 hierarchy): `{ name, px, use }[]` — 타입 스케일 표시용.

### 2. 컴포넌트 수정

- `FontOverviewDiagram.tsx`: 내부 상수 제거, `content/labs/font-lab/overview.json` import 후 동일 구조로 렌더링.
- 타입은 공용 또는 font-lab 전용 interface로 정의.

### 3. 검증

- Font Lab 페이지에서 오버뷰·타입 스케일·시맨틱 역할 표시가 기존과 동일한지 확인.
