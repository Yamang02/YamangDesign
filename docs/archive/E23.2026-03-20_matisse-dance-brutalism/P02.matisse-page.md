# P02: MatisseDanceII 페이지 구현

## 목표

브루탈리즘 사조로 마티스 《춤 II》 아트 페이지 3챕터를 구현한다.
P01에서 만든 공통 컴포넌트를 사용하며, Ch.2는 타이포그래피 인터랙션으로 자급자족 구현한다.

## 구현 상세

### 팔레트 상수

```ts
const MATISSE_PALETTE = [
  { name: 'Matisse Red',  hex: '#C8361A', role: 'Primary' },
  { name: 'Dance Green',  hex: '#2E6B35', role: 'Secondary' },
  { name: 'Sky Blue',     hex: '#1A4F8C', role: 'Accent' },
  { name: 'Cement',       hex: '#B0A89A', role: 'Neutral' },
  { name: 'Raw White',    hex: '#F2EFE8', role: 'Surface' },
  { name: 'Charcoal',     hex: '#1C1C1C', role: 'Dark' },
]
```

### 브루탈리즘 CSS 토큰

```css
/* 글라스모피즘 → 브루탈리즘 대체 규칙 */
backdrop-filter: none;                          /* 제거 */
background: <solid hex>;                        /* 솔리드 색상 */
border: 3px solid #1C1C1C;                     /* 하드 보더 */
border-radius: 0;                               /* 직각 */
box-shadow: 4px 4px 0 #1C1C1C;                 /* 오프셋 섀도우 */
```

### Ch.1 — The Painting

`ArtHeroStage` 컴포넌트 사용:
- `imageUrl`: `https://upload.wikimedia.org/wikipedia/commons/a/a7/Matissedance.jpg`
- `label`: `{ title: 'La Danse (II)', artist: 'Henri Matisse', medium: 'Oil on canvas', year: 1910 }`
- `palette`: MATISSE_PALETTE (상위 6색)
- 오버레이: 그라디언트 대신 하드 보더 프레임 (상하좌우 3px `#1C1C1C`)

### Ch.2 — The Impression (자급자족)

**`BrutalBackground.tsx`**
- 배경: `#1C1C1C` 단색
- 마티스 3색 솔리드 블록 3개 (Red·Green·Blue) 격자 배치, 고정 위치

**`BrutalTypographyQuote.tsx`**
- 화면 중앙 거대 타이포 (예: `"DANCE"` 또는 마티스 인용구)
- 각 글자를 `<span>`으로 분리, 마우스 위치에 따라 가까운 글자부터 Red→Green→Blue 색상 반응 (mousemove 이벤트)
- 인용구 카드:
  - `background: #C8361A`
  - `border: 3px solid #1C1C1C`
  - `box-shadow: 6px 6px 0 #1C1C1C`
  - `border-radius: 0`
- 인용구 텍스트: *"Creativity takes courage."* — Henri Matisse

### Ch.3 — The Application

`ArtApplicationSection` 컴포넌트 사용 (`themeStyle="brutal"`):

```ts
buttons: [
  { label: 'Primary',   hex: '#C8361A', textHex: '#F2EFE8' },
  { label: 'Secondary', hex: '#2E6B35', textHex: '#F2EFE8' },
  { label: 'Accent',    hex: '#1A4F8C', textHex: '#F2EFE8' },
  { label: 'Ghost',     hex: 'transparent', textHex: '#1C1C1C' },
]

typography: [
  { level: 'Display', text: 'La Danse',         hex: '#C8361A' },
  { level: 'Body',    text: 'Creativity takes courage.', hex: '#1C1C1C' },
  { level: 'Caption', text: 'Henri Matisse, 1910', hex: '#B0A89A' },
]

artCard: {
  imageUrl: 'https://upload.wikimedia.org/wikipedia/commons/a/a7/Matissedance.jpg',
  title: 'La Danse (II)',
  artist: 'Henri Matisse',
  year: 1910,
}
```
