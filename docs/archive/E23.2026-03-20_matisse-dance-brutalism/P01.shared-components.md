# P01: 공통 컴포넌트 추출

## 목표

Monet 페이지에서 실제로 겹치는 컴포넌트를 `_shared/`로 이동하고,
HeroStage · ApplicationSection 공통 레이아웃 컴포넌트를 신규 생성한다.
MonetWaterLilies를 새 공통 컴포넌트 기반으로 업데이트해 기존 동작이 깨지지 않는지 확인한다.

## 구현 상세

### 이동 (Monet → _shared)

| 파일 | 현재 위치 | 이동 위치 |
|------|-----------|-----------|
| `MuseumLabel.tsx` | `MonetWaterLilies/components/` | `_shared/` |
| `MuseumLabel.module.css` | `MonetWaterLilies/components/` | `_shared/` |
| `PaletteSwatchBar.tsx` | `MonetWaterLilies/components/` | `_shared/` |
| `PaletteSwatchBar.module.css` | `MonetWaterLilies/components/` | `_shared/` |

props 인터페이스 변경 없음. import 경로만 업데이트.

### 신규 생성

**`_shared/ArtHeroStage.tsx`**
```tsx
interface ArtHeroStageProps {
  imageUrl: string
  imageAlt: string
  label: {
    title: string
    subtitle?: string
    artist: string
    medium: string
    year: number
  }
  palette: { name: string; hex: string }[]
}
```
- 이미지 히어로(65vh, min 480px) + 오버레이 슬롯
- 우상단 MuseumLabel, 좌하단 PaletteSwatchBar 배치
- 오버레이 스타일은 CSS 변수(`--art-hero-overlay`)로 주입 가능하게 설계

**`_shared/ArtApplicationSection.tsx`**
```tsx
interface ArtPaletteItem { name: string; hex: string; role: string }
interface ArtButtonItem { label: string; hex: string; textHex: string }
interface ArtTypographyItem { level: string; text: string; hex: string }
interface ArtCardMeta { imageUrl: string; title: string; artist: string; year: number }

interface ArtApplicationSectionProps {
  palette: ArtPaletteItem[]
  buttons: ArtButtonItem[]
  typography: ArtTypographyItem[]
  artCard: ArtCardMeta
  themeStyle?: 'glass' | 'brutal'  // 컨테이너 스타일 토글
}
```
- 4블록(Color Tokens, Buttons, Typography, Art Card) 공통 레이아웃
- `themeStyle` prop으로 glassmorphism vs brutalism 컨테이너 스타일 분기

### 수정

**`MonetWaterLilies.tsx`**
- 직접 구현 → `ArtHeroStage`, `ArtApplicationSection` import로 교체
- import 경로: `../_shared/MuseumLabel` → 제거 (ArtHeroStage 내부로 이동)
