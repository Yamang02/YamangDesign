# P02: Service Context 재정의

## 목표

Service 페이지를 "실제 서비스 페이지 목업"에 집중시킨다.
Build와 역할이 겹치는 Component Set 섹션을 제거하고 Page Preview를 강화한다.

## 구현 상세

### Component Set 섹션 제거

`ServiceContext.tsx`에서 `<LabSection title="Component Set" ...>` 블록 및 관련 상수(MOLECULES, ORGANISMS import) 제거.
TOC에서 `component-set` 항목 제거.

### Page Preview 강화

현재 Landing 스타일 1종 → 여러 페이지 유형으로 확장.

| 탭 | 내용 |
|----|------|
| Landing | 현행 히어로 + 카드 그리드 + 폼 (유지) |
| Dashboard | 통계 카드 + 테이블 목업 |
| Article | 타이포그래피 중심 콘텐츠 레이아웃 |

탭 전환 UI 추가. 모든 탭은 동일 `themeVars`를 `data-context="preview"`로 주입.

### Controls 개선

`glassmorphism` 등 신규 스타일 프리셋이 드롭다운에 자동 반영되도록
style 옵션을 preset registry에서 동적 파생 (GlobalSettingsModal P01과 동일 패턴).
