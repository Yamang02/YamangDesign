# P03: Build Theme 주입

## 목표

Build 페이지(Atoms / Molecules / Organisms) 상단에 경량 Controls를 추가하여
"이 컴포넌트가 다른 테마에서 어떻게 보이는가"를 탐구할 수 있게 한다.

## 구현 상세

### Controls 범위

Service의 풀 Controls(Palette + System + Neutral + Style + Font)와 달리
Build는 컴포넌트 구조 탐구가 목적이므로 **Palette + Style** 두 축만 제공.

```
[ Palette: sage ▼ ]  [ Style: minimal ▼ ]
```

### 상태 관리

- Build 페이지 내 로컬 state (`useState`)
- 글로벌 ThemeProvider와 독립적 — 글로벌 테마를 변경하지 않음
- `getThemeVariables(palette, style)` 로 `themeVars` 계산
- 컴포넌트 렌더 영역에 `data-context="preview"` + `style={themeVars}` 주입

### 적용 범위

Atoms / Molecules / Organisms 세 페이지 모두 동일 패턴 적용.
Controls는 각 페이지 최상단 `<LabSection>` 이전에 위치.
