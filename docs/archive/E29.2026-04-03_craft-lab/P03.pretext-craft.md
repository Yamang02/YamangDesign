# E29 P03: Pretext 텍스트 레이아웃 Craft 구현

## 목표

- CraftLab에 `@chenglou/pretext` 텍스트 레이아웃 기법 섹션이 존재한다
- 격자 문자열을 `prepareWithSegments`로 세그먼트화한 뒤 `layoutWithLines`로 줄 단위 레이아웃을 얻고, Canvas 위에 텍스트가 실루엣 모양으로 채워지는 예시가 동작한다
- Golconda에서 쓴 기법이 독립 예시로 재현되어, 기법의 원리를 확인할 수 있다

## 구현 상세

### 접근 방법

Golconda의 `gentlemanSilhouette.ts`에서 pretext 사용 패턴을 추출해 standalone 컴포넌트로 재구성한다. 실루엣 모양은 단순한 원·사각형·삼각형 중 선택 가능하게 하여 기법의 핵심(모양 마스크 → 텍스트 채우기)이 명확히 드러나게 한다.

- `prepareWithSegments(block, font, { whiteSpace: 'pre-wrap' })` — 블록을 측정 가능한 세그먼트로 준비
- `layoutWithLines(prepared, maxWidth, lineHeight)` — 줄 배열·전체 높이 산출

사용자가 조작할 변수:
- 실루엣 모양 선택 (circle / square / triangle)
- 채움 텍스트 패턴 (짧은 문자열 입력)
- 폰트 크기 (6–14px 범위)
