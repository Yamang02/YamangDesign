# E29 P04: yamang-craft 스킬 작성

## 목표

- 레포 `.cursor/skills/yamang-craft/SKILL.md`가 존재하고, 새 Craft를 CraftLab에 추가하는 절차를 완전히 기술한다
- 스킬을 따라 새 craft를 추가할 때 누락 없이 동작하는 체크리스트가 포함된다
- `yamang-art-page-workflow` 스킬과 구조적으로 대응하여 동일한 기준으로 읽힌다

## 구현 상세

### 접근 방법

`yamang-art-page-workflow` 스킬(레포 `.cursor/skills`)을 기준 삼아 CraftLab 도메인에 맞게 작성한다. art의 "3챕터"에 대응하는 craft의 "3구성요소"를 정의한다:

| art 챕터 | craft 구성요소 |
|---|---|
| Ch.1 — The Painting | **설명** — 기법 이름, 한 문장 요약, 언제 쓰는가 |
| Ch.2 — The Impression | **라이브 예시** — 인터랙티브 Canvas 또는 DOM 데모 |
| Ch.3 — The Application | **코드 스니펫** — 핵심 코드 표시, 복사 가능 |

### 스킬이 다뤄야 할 내용

1. **아키텍처 규칙**: 디렉토리 구조 (`crafts/{CraftName}/`), CraftLab.tsx 마운트 방식, DS 변수 사용 원칙
2. **3구성요소 규격**: 각 구성요소의 필수 props/슬롯 정의
3. **라우팅 통합 체크리스트**: 새 craft는 `CraftLab`에 섹션으로 추가; `PageName`/`nav`는 craft 랩 단일 진입으로 이미 연결됨
4. **체크리스트**: 새 craft 추가 시 완료 기준 항목
