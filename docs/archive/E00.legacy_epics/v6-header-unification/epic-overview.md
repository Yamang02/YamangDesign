# Epic: 헤더 스타일 통일 및 전역 설정 패널 구축

## 개요

YamangDesign 디자인 시스템의 헤더를 AI_Portfolio 스타일로 통일하고, 분산되어 있는 전역 설정 기능을 하나의 통합된 패널/모달로 관리하는 구조로 개선합니다.

## 배경 및 목표

### 현재 상태

**헤더 비교:**
| 항목 | YamangDesign | AI_Portfolio |
|------|--------------|--------------|
| 브랜딩 | 텍스트 "Yamang Design" | 한글 "야망솔루션" |
| 네비게이션 | 드롭다운 기반 카테고리 | 아이콘 기반 + 툴팁 |
| 레이아웃 | 3분할 (좌/중앙/우) | 2분할 (좌/우) |
| 모바일 대응 | 미구현 | 햄버거 메뉴 구현 |
| 설정 접근 | 컬러 에디터만 헤더에 위치 | 설정 아이콘 (admin) |

**전역 설정 현황:**
- 컬러 에디터: Navigation 컴포넌트 내 인라인 패널
- 스타일/팔레트 선택: Playground 페이지의 Select 드롭다운들
- 시스템 프리셋: Playground에서만 접근 가능
- **문제점**: 설정이 여러 곳에 분산, 발견하기 어려움

### 목표

1. **헤더 통일**: AI_Portfolio의 아이콘 기반 네비게이션 스타일 적용
2. **브랜딩 통일**: "야망디자인" 로고 사용 (동일 폰트, 현재 색상 유지)
3. **설정 통합**: 전역 설정 패널/모달로 모든 디자인 토큰 관리 중앙화

## 스토리 목록

### E01: 헤더 레이아웃 재구성
- AI_Portfolio 스타일의 2분할 레이아웃 적용
- "야망디자인" 브랜딩 로고 추가
- 현재 디자인 시스템 색상/폰트 유지

### E02: 아이콘 기반 네비게이션 구현
- 복잡한 드롭다운 메뉴를 아이콘 네비게이션으로 전환
- 디자인 시스템 특성에 맞는 메뉴 구조 설계
- 툴팁 지원

### E03: 전역 설정 모달 설계 및 구현
- 통합 설정 패널 컴포넌트 구축
- 팔레트, 스타일, 시스템 프리셋 통합 관리
- 설정 저장/불러오기 기능

## 아키텍처 고려사항

### 메뉴 구조 제안

디자인 시스템은 AI_Portfolio보다 복잡한 네비게이션이 필요합니다:

**방안 A: 2단계 아이콘 네비게이션**
```
[야망디자인]     [Pages▾] [Components] [Labs▾] [⚙️설정]
```
- Pages와 Labs는 클릭 시 드롭다운 표시
- Components는 단일 페이지로 직접 이동
- 설정 아이콘으로 전역 설정 모달 오픈

**방안 B: 메가 메뉴 스타일**
```
[야망디자인]     [🏠] [📦] [🧪] [⚙️]
                       ↓
              ┌─────────────┐
              │ Palette Lab │
              │ Style Lab   │
              │ Font Lab    │
              │ Playground  │
              └─────────────┘
```
- 모든 메뉴가 아이콘화
- 호버/클릭 시 서브메뉴 패널 표시

**✅ 확정: 방안 A** - AI_Portfolio 스타일 유지하면서 필요한 복잡도 수용

### 로고 폰트 (E01)
- **Griun PolSensibility** (AI_Portfolio 헤더와 동일)
- 디자인시스템 토큰: `--ds-font-display`

## 기술적 고려사항

### 재사용 가능한 기존 코드
- `ThemeProvider`: 전역 상태 관리 (이미 구현됨)
- `ColorPicker`, `PresetManager`: 팔레트 편집 UI
- `usePalettePresets`: 프리셋 저장/불러오기
- `HeaderIconButton` (AI_Portfolio): 아이콘 버튼 스타일 참조

### 새로 개발 필요
- `GlobalSettingsModal`: 통합 설정 모달 컴포넌트
- `IconNav`: 아이콘 기반 네비게이션 컴포넌트
- `NavMegaMenu`: 서브메뉴 드롭다운 (방안 A 선택 시)

### CSS 변수 체계
현재 디자인 시스템 토큰 유지:
- `--ds-color-*`: 색상 토큰
- `--ds-spacing-*`: 간격 토큰
- `--ds-text-*`: 타이포그래피 토큰
- `--ui-*`: UI 시맨틱 토큰

## 의존성

- CSS Modules 스타일링 방식 유지
