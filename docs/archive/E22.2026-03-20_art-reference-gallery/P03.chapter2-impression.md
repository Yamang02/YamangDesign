# P03: Chapter 2 — The Impression

## 목표

모네 원화 이미지를 사용하지 않고, 회화의 색채·빛·물결 심상을 순수 CSS/JS로 재구성한
인터랙티브 구간을 구현한다.

## 구현 상세

**접근 방법:**
Chapter 1 스크롤 이후 시작되는 독립 섹션. 배경은 CSS animated gradient로 수면의 빛 일렁임을
시뮬레이션하고, 마우스 인터랙션으로 파문 효과를 준다. 이미지 없이 팔레트 DNA만으로
"모네의 느낌"을 재현하는 것이 핵심 콘셉트.

**변경 파일 (신규 생성):**
- `src/app/pages/art/MonetWaterLilies/components/ImpressionBackground.tsx` — animated gradient 배경
- `src/app/pages/art/MonetWaterLilies/components/ImpressionBackground.module.css`
- `src/app/pages/art/MonetWaterLilies/components/RippleCanvas.tsx` — 마우스 파문 인터랙션
- `src/app/pages/art/MonetWaterLilies/components/RippleCanvas.module.css`
- `src/app/pages/art/MonetWaterLilies/components/ColorDNAMap.tsx` — 색상 scatter 시각화
- `src/app/pages/art/MonetWaterLilies/components/ColorDNAMap.module.css`
- `src/app/pages/art/MonetWaterLilies/components/ImpressionistQuote.tsx` — 단어 fade 타이포
- `src/app/pages/art/MonetWaterLilies/components/ImpressionistQuote.module.css`

**ImpressionBackground:**
- `@keyframes` 로 3개 radial-gradient가 8~12s 주기로 위치 이동
- 모네 팔레트 기반 색상: `rgba(45,80,22,0.4)`, `rgba(155,139,180,0.3)`, `rgba(232,228,213,0.15)`
- 전체 섹션 배경에 `position: absolute; inset: 0`

**RippleCanvas:**
- `<canvas>` 태그 사용, `mousemove` 이벤트 시 클릭 지점에서 원형 파문 애니메이션
- 파문 색상: 모네 팔레트 중 Water Blue / Teal 계열 반투명
- 복수 파문 동시 존재 가능 (배열로 관리, requestAnimationFrame 루프)

**ColorDNAMap:**
- 8색을 2D scatter로 배치: x축 = 채도(S), y축 = 명도(L) (HSL 변환)
- 각 dot: `32×32px` 원형, 해당 색상으로 채움
- hover 시 색상명 + HEX 툴팁 (glass 스타일, `--ds-spacing-*` 참조)
- SVG 또는 순수 div positioning으로 구현

**ImpressionistQuote:**
- 인용구: *"I perhaps owe having become a painter to flowers."* — Claude Monet
- 단어 단위로 `animation-delay` 차등 적용, `opacity: 0 → 1` + `filter: blur(4px) → blur(0)`
- 전체 루프: 페이지 진입 후 1회 재생
