# P02: token-section-restyle

## 목표
모달 내 Design Tokens / Key tokens 섹션의 시각 스타일을 DetailPanel과 어울리도록 리디자인한다.

## 구현 상세

### 스타일 방향 (DetailPanel 느낌)
**참조**: `src/components/DetailPanel/DetailPanel.module.css`

| 현재 | 변경 |
|------|------|
| 카테고리 타이틀에 좌측 테두리 + 배경색 강조 | 배경·테두리 없이 muted 소문자 텍스트 |
| 토큰 행: 2컬럼 그리드 (280px name \| 1fr value) | flex column: 이름 위, 값 아래 |
| 값: 작은 pill (background: shell-bg-muted) | inline code, 최소 패딩 |
| 카테고리 간격 넓음 | 섹션 간격 압축 |

**파일**: `src/pages/build/Components/Components.module.css`

수정 클래스:
- `.tokenGroupTitle` — `background`, `border-left`, `padding` 제거. `font-size: var(--ds-text-xs)`, `color: var(--shell-text-muted)`, `text-transform: uppercase`, `letter-spacing: 0.05em` 유지
- `.tokenRow` — `display: grid` → `display: flex; flex-direction: column; gap: var(--ds-spacing-1)`. `border-bottom` 유지 (행 구분용)
- `.tokenMeta` — 변경 없음 (이름 + 레이블 세로 배치 유지)
- `.tokenValue` — `background` 제거 또는 더 subtle하게. `padding` 최소화 (2px 0)
- `.tokenRowLong` — flex column이 기본이 되므로 별도 처리 불필요 시 제거

### Molecules / Organisms 섹션 타이틀 변경
**파일**: `src/pages/build/Molecules.tsx`, `src/pages/build/Organisms.tsx`

```tsx
// 변경 전
<h2 className={styles.showcaseModalSectionTitle}>Design tokens</h2>
// 변경 후
<h2 className={styles.showcaseModalSectionTitle}>Key tokens</h2>
```
Atoms는 `"Design tokens"` 유지.

### Organisms build-content.ts — shell 토큰 제거
**파일**: `src/constants/build-content.ts`

`header-bar` tokens에서 `{ token: '--shell-bg-base', label: '헤더 배경' }` 제거.
`sidebar-nav` tokens에서 `{ token: '--shell-bg-surface', label: '사이드바 배경' }` 제거.

Key tokens로서 의미 있는 `--ds-*` 토큰만 잔존.
