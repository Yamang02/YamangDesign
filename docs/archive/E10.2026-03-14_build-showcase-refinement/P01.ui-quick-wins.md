# P01: ui-quick-wins

## 목표
구조·스타일 변경 없이 처리 가능한 UI 수정 3가지를 완료한다.

## 구현 상세

### ① Atoms — Examples 섹션 제거
**파일**: `src/pages/build/Atoms/Atoms.tsx`

제거 대상:
- `EXAMPLES_IDS` 상수 선언
- `tocItems` 배열에서 `{ id: 'form-example', label: formExample.title }` 항목
- `VARIANT_COUNTS` 객체에서 `'form-example': 1` 엔트리
- `getCardPreview` switch 문에서 `case 'form-example'` 분기
- JSX의 `<section className={styles.showcaseExamples}>…</section>` 블록 전체

`formExample` import가 다른 곳에서 쓰이지 않으면 import도 제거.

### ② Organisms — 1행 레이아웃
**파일**: `src/pages/build/Organisms.tsx`

`showcaseGrid` div에 `style={{ gridTemplateColumns: '1fr' }}` 추가.
별도 CSS 클래스 신설 없이 인라인으로 처리.

### ③ ComponentCard — composedOfLabel prop 추가
**파일**: `src/components/ComponentCard/ComponentCard.tsx`

```ts
composedOfLabel?: string;  // 기본값 "Atoms:"
```
렌더링: `<span className={styles.composedOfLabel}>{composedOfLabel ?? 'Atoms:'}</span>`

**파일**: `src/pages/build/Organisms.tsx`

Organism 카드에 `composedOfLabel="Molecules:"` prop 전달.
